OuFNis HW
************

OuFNis HW is a collection of tools aims at helping developers in their usually time consuming tasks. The following tools are present at the moment:

* A set of tools aimed for the SMD line in order to recover the tagged production package, check them and get files that are production ready
* A set of tools to create the production files directly from the Altium outputs
* A set of calculators, like optocoupler regimes
* Some planification tools (not released yet)
